El sistema desarrollado comprende tanto hardware como software y se orienta al aseguramiento de la calidad de la leche producida por pequeños ganaderos, con el fin de garantizar su comercialización en mercados con altos estándares. La incorporación del sistema permite un monitoreo continuo de variables sanitarias clave, asegurando la inocuidad de la leche. Esta tecnología se integra en las fases de ordeño, almacenamiento, transporte y entrega, garantizando que la producción sea adecuada para su mezcla en los tanques industriales de enfriamiento y para el procesamiento industrial y comercialización final.

De manera general, el sistema recopila los datos a partir de sensores que monitorean variables críticas. Estos valores son almacenados y procesados temporalmente en el microcontrolador. Según la conectividad disponible, la información permanece en almacenamiento local (modo offline) o se envía automáticamente a una base de datos en la nube. En la nube se ejecuta el análisis de las variables críticas, generando alertas que advierten al usuario si los valores se encuentran dentro o fuera de los rangos aceptables.

Durante el diseño e implementación del sistema se abordan varios desafíos. En cuanto al hardware, se garantiza la conectividad en entornos de baja cobertura y la seguridad de los componentes electrónicos, que se encuentran en contacto cercano con la leche. Para ello, se diseña una tapa para las cantinas, con fijaciones y sellado hermético, que protege tanto el producto como los componentes frente a contaminantes externos, desprendimientos o daños ambientales.

Respecto al software, se implementa un esquema de almacenamiento local en el microcontrolador para asegurar la recolección continua de datos, incluso en ausencia de conexión. Al restablecerse la conectividad, la información se transfiere de forma automática a la base de datos en la nube, lo que permite a los usuarios acceder a ella de manera confiable.
